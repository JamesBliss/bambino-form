# Form

The `<Form />` component is the most important element of the puzzle, it controls the flow and logic of all that reside within it.

`<Form />` is the place you set a parent schema, initial data set and the sumbit fuction.

## Props

<Props of={ Form } />

## Basic usage

<Playground>
  <Form onSubmit={ data => alert(JSON.stringify(data, null, 2)) }>
    <Input
      name='name'
      placeholder='Enter name'
    />
    <br />
    {/* This is required because docz doesn't support form submission ü§¶‚Äç */}
    <Context.Consumer>
      { ({ handleSubmit }) => (
        <button onClick={ (e) => handleSubmit(e) }>Submit name </button>
      )}
    </Context.Consumer>
  </Form>
</Playground>
