# Scope

The `<Scope />` component is a magical component which allows you to nest fields!

## Props

<Props of={ Scope } />

## Basic usage

<Playground>
  <Form onSubmit={ data => alert(JSON.stringify(data, null, 2)) }>
    <Input
      name='name'
      placeholder='Name'
    />
    <br />
    <Scope path='child'>
      <Input
        name='child_field'
        placeholder='Child field'
      />
    </Scope>
    <br />
    {/* This is required because docz doesn't support form submission ü§¶‚Äç */}
    <Context.Consumer>
      { ({ handleSubmit }) => (
        <button onClick={ (e) => handleSubmit(e) }>Submit name </button>
      )}
    </Context.Consumer>
  </Form>
</Playground>
